# Netmaze-Project
Multi-tier data processing platform using AWS and Linux
## Project Overview
NetMaze is a multi-tier web application deployed on AWS. This project demonstrates how to host a highly available, scalable, and secure application using AWS services and Linux administration.

## AWS Services Used
- Amazon EC2
- Application Load Balancer (ALB)
- Auto Scaling Group (ASG)
- Amazon VPC
- IAM
- Amazon S3
- CloudFront
- Route 53
- Security Groups

## Linux Tasks
- Apache Web Server Installation
- User and Permission Management
- File System Management
- Shell Scripting
- Service Monitoring

## Project Architecture
Client → Route 53 → CloudFront → ALB → EC2 (Apache) → Auto Scaling Group

## Features
- High Availability
- Automatic Scaling
- Secure Network Design
- Load Balancing
- Static Content Delivery using CloudFront

## Skills Demonstrated
- AWS Cloud
- Linux Administration
- Networking
- Web Server Configuration
- Troubleshooting
